Introduction

ðŸ‘‰ Bubble Sort is a simple sorting algorithm that is used to sort a list or array of elements by comparing each pair of adjacent elements and swapping them if they are in the wrong order. This process is repeated for each element in the list until no more swaps are required, indicating that the list is sorted.

How Bubble Sort Works

  1. Start from the first element (index 0) and compare it with the next element (index 1).
  2. If the first element is greater than the second element, swap them.
  3. Move to the next pair of elements (index 1 and index 2) and repeat the comparison and swapping.
  4. Continue this process, moving through the entire list, until no more swaps are needed.
  5. The largest element will "bubble" up to the end of the list in each pass.
  6. Repeat these steps for each element in the list until the entire list is sorted.

Pseudo Code

function bubbleSort(arr):
  n = length of arr
  for i from 0 to n-1:
    for j from 0 to n-i-1:
      if arr[j] > arr[j+1]:
        swap arr[j] and arr[j+1]

Time Complexity

The time complexity of Bubble Sort is O(n^2), where 'n' is the number of elements in the array. It is not very efficient for large lists, as it has to make multiple passes through the list, resulting in quadratic time complexity.

Space Complexity

The space complexity of Bubble Sort is O(1) because it doesn't use any additional memory for sorting. It sorts the elements in-place.
